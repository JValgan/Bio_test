# Quiz Biologia - Selectivitat PAU 2025

Quiz interactiu amb 100 preguntes de nivell de 2n de Batxillerat per preparar la selectivitat.

## Temàtiques
- Biomolècules
- Metabolisme
- Genètica
- Microorganismes i virus
- Immunologia
- Biotecnologia
- Evolució

## Funcionalitats
- Escull el nombre de preguntes (1-100)
- Ordre aleatori de preguntes cada partida
- Ordre aleatori de les opcions de resposta
- Correcció automàtica amb retroacció
- Puntuació final
